Aether Design Admin template generated by `create-aether-admin`.

## Scripts

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm build
```

## Included

- Vue 3, TypeScript, Vite
- Element Plus with Aether Design visual overrides
- Pinia and Vue Router
- Admin layout with multi-level sidebar
- Dashboard, list, detail, edit, settings and design system pages
- Business components for users, departments, permissions, workflow and admin tables
