# create-aether-admin

[简体中文](./README.zh-CN.md)

Create an Aether Design Admin Vue 3 + Element Plus starter.

```bash
npm create aether-admin@latest my-admin
```

Or use the short alias after global install:

```bash
npm i -g create-aether-admin
cea my-admin
```

Then:

```bash
cd my-admin
pnpm install
pnpm dev
```
