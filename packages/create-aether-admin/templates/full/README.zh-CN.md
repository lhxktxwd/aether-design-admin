# __PROJECT_NAME__

[English](./README.md)

由 `create-aether-admin` 生成的 Aether Design Admin 项目模板。

## 常用命令

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm build
```

## 包含内容

- Vue 3, TypeScript, Vite
- Element Plus + Aether Design 视觉覆盖
- Pinia 和 Vue Router
- 支持多级子菜单的 Admin Layout
- Dashboard、列表、详情、编辑、设置、设计系统页面
- 用户选择、部门选择、权限矩阵、状态流转、复杂表格组件
